# Task 3. Test Plan for "How I learn" section

## 1. Introduction
The purpose of this Test Plan is to define the testing strategy for the "How we learn" section of the School 21 website.

## 2. Scope of Work
Testing will be focused on:
1. Content accuracy (text and icons).
2. Navigation elements (links and buttons).
3. Responsiveness of the page layout.

## 3. Testing Type
1. Functional Testing: To verify buttons, links, and forms.
2. UI/UX Testing: To check the visual layout and user experience.

## 4. Test Environment
1. Browsers: Google Chrome, Mozilla Firefox, Safari.
2. Devices: Desktop (1920x1080).

## 5. Entry and Exit Criteria
1. Entry: The "How we learn" page is live and accessible.
2. Exit: All 10 check-list items (from Task 2 and 4) are tested, and bugs are reported.

## 6. Deliverables
1. Test Plan (this document).
2. Check-lists and Test Cases.
3. Final Test Report.
